# Minimal GitHub Pages Site

This folder contains a simple static profile site suitable for a personal `github.io` page.

## Files

- `index.html`: landing page
- `styles.css`: shared styling
- `posts/on-device-ai-android.html`: blog post about the Android llama.cpp project

## Publish

1. Create a new repository named `<your-github-username>.github.io`
2. Copy the contents of this folder into that repository
3. Commit and push to the `main` branch
4. GitHub Pages will serve it automatically

## Personalize

Update these placeholders in `index.html`:

- `your-github-username`
- `your-linkedin-handle`
- `you@example.com`

You should also adjust the visible name and intro text if needed.
